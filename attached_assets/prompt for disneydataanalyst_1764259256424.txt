Comprehensive Prompt for Personal Portfolio Website
Project Overview
Design and develop a modern, professional portfolio website for Gurdeep Singh, a Data/Business Analyst with expertise in SQL, Python, Excel, and data visualization tools. The site should showcase his analytical projects, professional experience, and technical capabilities while maintaining a clean, data-driven aesthetic inspired by the Pixar Cars universe (specifically the character Mater from the Disney Cars franchise).
________________


Brand Identity & Design System
Visual Theme
* Primary Inspiration: Disney Pixar's "Cars" franchise — specifically Mater's friendly, approachable, yet skilled personality
* Color Palette:
   * Primary: Rust brown/burnt orange (#C47335) — represents Mater's iconic color
   * Secondary: Deep teal/turquoise (#4A9B9B) — professional data analytics feel
   * Accent: Lightning McQueen red (#E31E24) — for CTAs and highlights
   * Neutral: Warm grays (#F5F5F0, #3A3A3A) — backgrounds and text
   * Success/Data: Radiator Springs sky blue (#87CEEB) — for data visualizations
Typography
* Headings: "Bebas Neue" or "Racing Sans One" (bold, automotive-inspired)
* Body Text: "Inter" or "Open Sans" (clean, highly readable)
* Code/Data: "JetBrains Mono" or "Fira Code" (for technical sections)
Design Principles
* Clean, minimalist layouts with ample white space
* Subtle automotive/route-inspired decorative elements (dotted lines as section dividers, route markers)
* Data visualization aesthetics — charts, graphs as background patterns
* Mobile-first responsive design
* Accessibility compliance (WCAG 2.1 AA)
* Fast loading times (<3 seconds)
________________


Technical Specifications
Tech Stack
* Frontend Framework: React.js with Next.js (for SEO and performance)
* Styling: Tailwind CSS with custom theme configuration
* Animations: Framer Motion for smooth transitions
* Icons: Lucide React or Heroicons
* Deployment: Vercel or Netlify
* Analytics: Google Analytics 4
Performance Requirements
* Lighthouse score: 90+ across all metrics
* Core Web Vitals: LCP <2.5s, FID <100ms, CLS <0.1
* Image optimization: WebP format with lazy loading
* Code splitting and tree shaking

Accessibility Considerations
* Semantic HTML5: Proper heading hierarchy, nav, section, article tags
* ARIA Labels: All interactive elements properly labeled
* Keyboard Navigation: Full site navigable via keyboard
* Focus Indicators: Clear, high-contrast focus states
* Alt Text: Descriptive alt text for all images
* Color Contrast: WCAG AA compliant (4.5:1 for body text, 3:1 for large text)
* Screen Reader Testing: Test with NVDA/JAWS
* Responsive Text: Font sizes scale appropriately, no horizontal scrolling

Animation Guidelines
Page Load
* Fade-in + slight upward motion for each section
* Staggered delays (0.1s between elements)
* Duration: 0.6s with ease-out curve
Hover States
* Scale: 1.05 for cards
* Shadow: Subtle elevation increase
* Color: Slight brightness/saturation boost
* Duration: 0.3s
Scroll Animations
* Intersection Observer API for performance
* Fade-in + slide-up for sections
* Parallax effects for background elements
* Progress bar for reading position
Transitions
* Page transitions: Slide + fade (0.4s)
* Modal open/close: Scale + fade (0.3s)
* Theme toggle: Color transition (0.5s)
